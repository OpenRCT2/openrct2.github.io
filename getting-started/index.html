Install OpenRCT2 for Arch Linux

  1. 1

    Install OpenRCT2

    You can obtain the latest release or develop version of OpenRCT2 from the ARU.
    yaourt -S openrct2
  2. 2

    Find your RCT2 installation

    openrct2 set-rct2 /path/to/rct2-install
