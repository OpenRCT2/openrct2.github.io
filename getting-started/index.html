Getting Started

Select your environment

Install OpenRCT2 for Windows (32 bit)

  1. 1

    Install OpenRCT2

    You can obtain the latest release or develop version of OpenRCT2 from OpenRCT2.

    Download latest release
    Download latest development build
  2. 2

    Find your RCT2 installation

    When you launch OpenRCT2 for the first time, OpenRCT2 will try to find your RCT2 installation. It will check the default locations that RCT2 is typically installed for all known releases. If it is unable to find it, or you have installed it to a different location, OpenRCT2 will show a dialog allowing you to select the directory containing your RCT2 files.

Install OpenRCT2 for Arch Linux

  1. 1

    Install OpenRCT2

    You can obtain the latest release or develop version of OpenRCT2 from the AUR.
    yaourt -S openrct2
  2. 2

    Find your RCT2 installation

    openrct2 set-rct2 /path/to/rct2-install

Install OpenRCT2 for Alpine Linux

  1. 1

    Install dependencies

    You will need the dependencies to build the game - there is no package in apk for the game as of yet.
    apk add git gcc g++ jansson-dev libzip-dev curl-dev libressl-dev sdl2-dev fontconfig-dev sdl2_ttf-dev fts-dev
  2. 2

    Build the game

    git clone https://github.com/OpenRCT2/OpenRCT2.git && cd ./OpenRCT2 && mkdir build && cd build && cmake ../ && make
  3. 3

    Set up the files

    cp -r ../data/ ./data/ && make g2 && mv ./g2.dat ./data/g2.dat
  5. 4

    Find your RCT2 installation

    openrct2 set-rct2 /path/to/rct2-install
